# Mechanical CAD Portfolio

This repository contains a collection of CAD projects created using **Fusion 360** and **Onshape**.  
It includes tutorials for skill-building, original mechanical designs, FRC-related assemblies, and high-quality renderings.

---

## Folder Structure

tutorials/ → CAD practice and guided exercises
original-projects/ → My own designs and mechanical concepts
renderings/ → Best project renderings (PNG, JPG)
exports/ → STEP files for all major parts

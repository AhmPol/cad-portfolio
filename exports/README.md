# CAD Exports

This folder contains all exported files for easy viewing and sharing.  

## File Types

- **STEP** (.step) → Universal CAD format, viewable in Fusion 360, Onshape, SolidWorks, FreeCAD  
- **STL** (.stl) → 3D printing format  
- **Images** (.png/.jpg) → High-quality renders, isometric, exploded, and orthographic views

## Usage

- Open STEP files in your preferred CAD software.  
- Images can be used for presentations or portfolio screenshots.
